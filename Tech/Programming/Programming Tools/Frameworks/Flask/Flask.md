Flask is a minimalist [[Python]] [[Framework]] used to create [Web](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FMISCELLANEOUS%2FWeb) Applications

It's often cited as a [[MicroFramework]] as It doesn't require particular tools or [Lib](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FMISCELLANEOUS%2FLib)
