webSocket is a [[Protocol]] that supports [[Full-Duplex]] communication through a single [TCP-IP](obsidian://open?vault=Obsidian&file=Tech%2FNetwork%2FPROTOCOLS%2FTCP-IP) connection.

It's used in client <--> [Server](obsidian://open?vault=Obsidian&file=Tech%2FHardware%2FConcepts%2FServer) communication.
