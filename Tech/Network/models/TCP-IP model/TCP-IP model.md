a set of rules to determine how computers should connect in the internet

Visual representation:
+------------------------------------------+
|           Application          |  Layer 4
+------------------------------------------+
|            Transport            |  Layer 3
+------------------------------------------+
|               Internet            |  Layer 2
+------------------------------------------+
|        Network access     |  Layer 1
+------------------------------------------+


## Application
End user layer, defines how data has be sent
Examples: HTTP
## Transport
Stablish connection, redirects to correct process in the host
Examples: TCP
## Internet
Responsible for discovering paths in the network and packet sending from host to host
Examples: IP 
## Network Access
Deals with the physical connection between devices
