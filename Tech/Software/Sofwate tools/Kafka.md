Kafka is an [[Open-Source]] [[Asynchronous]] [Messaging Service](Messaging%20Service.md) created by [[Apache]], It's written in [Java](obsidian://open?vault=Obsidian&file=Tech%2FProgramming%2FProgramming%20Languages%2FJava%2FJava) and [[Scala]]



Kafka is used to treat [Data](obsidian://open?vault=Obsidian&file=Tech%2FDatabase%2FData) in real time

Can be used to test and run [[Micro Services]] as well as [API](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FBACK-END%2FAPI%2FAPI)

Kafka doesn't have a single use
Uses:
1. Message [[Queue]]
2. Real time streaming
3. [Log](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FMISCELLANEOUS%2FLog) aggregation
