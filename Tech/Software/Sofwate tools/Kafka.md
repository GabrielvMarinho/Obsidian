Kafka is an [Open-Source](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FMISCELLANEOUS%2FOpen-Source) [Asynchronous](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FMISCELLANEOUS%2FAsynchronous) [Messaging Service](Messaging%20Service.md) created by [[Apache]], It's written in [Java](obsidian://open?vault=Obsidian&file=Tech%2FProgramming%2FProgramming%20Languages%2FJava%2FJava) and [Scala](obsidian://open?vault=Obsidian&file=Tech%2FProgramming%2FProgramming%20Languages%2FScala%2FScala)



Kafka is used to treat [Data](obsidian://open?vault=Obsidian&file=Tech%2FDatabase%2FData) in real time

Can be used to test and run [Micro Services](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FSoftware%20Arquitecture%2FPatterns%20of%20software%2FMicro%20Services%2FMicro%20Services) as well as [API](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FBACK-END%2FAPI%2FAPI)

Kafka doesn't have a single use
Uses:
1. Message [Queue](obsidian://open?vault=Obsidian&file=Tech%2FDataScience%2FData%20Structure%2FQueue)
2. Real time streaming
3. [Log](obsidian://open?vault=Obsidian&file=Tech%2FSoftware%2FConcepts%2FMISCELLANEOUS%2FLog) aggregation
