## Annotations

They are used to control and configurate the behaviour of an app [[Function]]s
##### @Component 
[[Class]]-level annotation that let's spring auto-detect these classes for [[Dependency Injections]]
##### @Service
contains the app logic
##### @Configuration
the [[Class]] contains spring configurations
##### @Bean
Define a bean inside a configuration [[Class]]
##### @Autowired
Injects [[Dependencies]] automatically managed by spring
